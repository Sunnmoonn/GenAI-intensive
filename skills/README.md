# Skills для Claude Code

Готовые скиллы для использования в Claude Code. Каждая папка содержит `SKILL.md` — установи скилл командой или скопируй файл в `~/.claude/skills/<название>/SKILL.md`.

---

## Яндекс Директ

| Скилл | Описание |
|-------|----------|
| [YD-find-segments](./YD-find-segments/SKILL.md) | Находит семантические сегменты для рекламных кампаний. Строит психологические слои аудитории, seed-фразы, анализирует Wordstat через API, выдаёт таблицу групп с ключами и минус-словами |
| [директ-апи](./директ-апи/SKILL.md) | Создаёт кампании Яндекс Директ через API v5: Поиск (SKAG-lite, WB_MAXIMUM_CLICKS) и РСЯ (AVERAGE_CPC) с фото и видео |

## Контент и медиа

| Скилл | Описание |
|-------|----------|
| [презентация](./презентация/SKILL.md) | Генерирует Reveal.js HTML-презентации из текста или заметок. 4 стиля (Minimalism, Dark, Neobrutalism, Avant), 3 уровня насыщенности, 12 макетов слайдов |
| [изобразика](./изобразика/SKILL.md) | Генерирует изображения через kie.ai API — Flux, Ideogram, GPT-4o, Nano Banana, Seedream, Grok и другие модели |
| [pdf-to-md](./pdf-to-md/SKILL.md) | Конвертирует PDF, PPTX, DOCX, XLSX, HTML и изображения в Markdown через MarkItDown (Microsoft) |

## Анализ аудитории

| Скилл | Описание |
|-------|----------|
| [ца-прогон](./ца-прогон/SKILL.md) | Прогоняет артефакт (лендинг, пост, продукт) через симулированную целевую аудиторию. Генерирует аватары, запускает агентов параллельно, собирает фидбек и рекомендации |

## Логирование и отчёты

| Скилл | Описание |
|-------|----------|
| [фиксируй](./фиксируй/SKILL.md) | Логирует результаты сессии Claude Code в SESSION_LOG.md — без вопросов и пауз |
| [фиксируй-неделю](./фиксируй-неделю/SKILL.md) | Суммаризирует сессии за неделю и дописывает итог в WEEKLY_LOG.md |
| [daily](./daily/SKILL.md) | Генерирует daily-отчёт и HTML-дашборд из сессий Claude Code |
