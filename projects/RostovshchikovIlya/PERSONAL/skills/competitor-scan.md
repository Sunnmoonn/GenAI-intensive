# competitor-scan

**Когда триггерится:** при появлении нового игрока на рынке автоматизации паркинга или при обновлении информации о существующем конкуренте

**Что делает:** собирает структурированную карточку конкурента по слоям — технология, ЦА, цена, слабые места, угроза для Easy Parking

**Контекст, который подкладывает:**
- Список текущих конкурентов: Паркоматика, Tele2 СмартПаркинг, Парковки России, Яндекс.Карты
- Наши отличия: USB (не OBD-II), B2C+B2B, подписка фиксированная, точность до 2м
- Шаблон карточки конкурента

## Шаблон карточки

```markdown
## [Название] — конкурентный анализ [дата]

**Тип:** информационный / полуавтомат / автомат
**Сегмент:** B2C / B2B / оба
**Технология:** ...
**Цена:** ...
**Покрытие:** ...

### Сильные стороны
- ...

### Слабые стороны
- ...

### Угроза для Easy Parking (1–5)
Оценка: ...
Почему: ...

### Как отстроиться
- ...
```

## Пример запуска

```
/competitor-scan

Название: Паркоматика
Источник: [ссылка или описание]
```
