# log-interview

**Когда триггерится:** после интервью или созвона с пользователем / потенциальным клиентом Easy Parking

**Что делает:** берёт сырые заметки или транскрипт разговора и превращает их в структурированную карточку user-interview с выделенными инсайтами, болями и цитатами

**Контекст, который подкладывает:**
- Шаблон карточки: сегмент → боль → инсайт → цитата → следующий шаг
- Правило «один инсайт = одна карточка»
- Формат Easy Parking: B2C (физлица) / B2B (транспортные компании)

## Шаблон

```markdown
## Интервью: [имя/роль] — [дата]

**Сегмент:** B2C / B2B — [описание]
**Контекст:** [где паркуется, как часто]

### Боли
- ...

### Инсайты
- ...

### Цитаты
> "..."

### Следующий шаг
- [ ] ...
```

## Пример запуска

```
/log-interview

[вставить заметки или транскрипт]
```
