# startup-answer

**Когда триггерится:** при заполнении заявки на акселератор, грант или конкурс стартапов

**Что делает:** пишет ответ на конкретный вопрос анкеты в нужной длине и тоне — с реальными данными Easy Parking, без воды

**Контекст, который подкладывает:**
- База фактов Easy Parking (проблема, решение, рынок, команда, traction)
- Типовые вопросы акселераторов: проблема / решение / рынок / конкуренты / команда / почему сейчас
- Правило: каждый ответ начинается с конкретного факта, а не с абстракции

## База фактов (всегда подкладывается)

- **Проблема:** 52% водителей забывают запустить парковку → штраф или эвакуация; 5,1 млн авто в Москве
- **Решение:** USB-блок телематики + подписка → автозапуск/завершение сессии без участия водителя
- **Цена:** устройство 4990₽ + 399₽/мес (vs Паркоматика — 16% от чека)
- **Точность:** до 2 метров в плотной застройке (GPS/ГЛОНАСС)
- **Traction:** юр. лицо 05.2024, MVP 10.2024, победители бизнес-акселератора 2023, Startup Village 2023, Студенческий стартап 2023
- **Цель 2026:** preseed 3 млн ₽, 150+ пользователей, MRR 100 000 ₽

## Пример запуска

```
/startup-answer

Вопрос: "Опишите проблему, которую решает ваш продукт (до 150 слов)"
Акселератор: Сколково
Тон: деловой, факты на первом месте
```
