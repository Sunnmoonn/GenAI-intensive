# weekly-review

**Когда триггерится:** каждое воскресенье вечером

**Что делает:** проводит ретро недели по двум проектам (Вникай + Easy Parking), закрывает незавершённые задачи, обновляет OKR и ставит 3 SMART-цели на следующую неделю

**Контекст, который подкладывает:**
- Текущие OKR по Easy Parking (preseed, B2B-контракты, MRR)
- Текущие OKR по Вникай (контент-пайплайн, деплой, уровни)
- Шаблон SMART-цели: конкретная / измеримая / достижимая / релевантная / срочная
- Итоги прошлых недель из vault

## Шаблон выхода

```markdown
## Неделя [дата]

### Сделано
- ...

### Не сделано → переносим / закрываем
- ...

### Решения
- ...

### 3 SMART-цели на следующую неделю
1. ...
2. ...
3. ...
```

## Пример запуска

```
/weekly-review

Сделал: ...
Не сделал: ...
```
