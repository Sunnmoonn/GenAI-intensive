## Мои AI-паттерны

### `product-thinking` — думать о продукте, а не о фичах
**Триггер:** есть идея или боль пользователя  
**Как использую:** описываю контекст → прошу разложить на слои (интерфейс, логика, данные, монетизация) → нахожу, где реальная ценность  
**Результат:** понимаю, что строить в первую очередь, а что — потом

### `vibe-coding` — от идеи до рабочего прототипа
**Триггер:** нужно проверить гипотезу быстро  
**Как использую:** описываю продукт на русском языке → Claude Code генерирует полный стек → итерирую через диалог  
**Результат:** за один сеанс получаю рабочий MVP без знания программирования

### `transcription-pipeline` — расшифровка видео
**Триггер:** есть видео, нужны субтитры с таймингами  
**Как использую:** Whisper (small model) → `.txt` с временными метками → Claude переводит на второй язык → подставляю в приложение  
**Результат:** полностью размеченные двуязычные субтитры за 3–5 минут на видео

## Как строился мой AI-контур за две недели

1. **Начал с нуля** — не знал программирования вообще
2. **Первый шаг** — установил Node.js, создал Vite + React проект через Claude Code
3. **Контентный слой** — загрузил 10 видео, расшифровал через Whisper, перевёл через Claude
4. **UI-итерации** — менял дизайн через диалог ("сделай темнее", "убери стрелки", "добавь свайпы")
5. **Инфраструктура** — GitHub, деплой — всё через пошаговые инструкции от Claude

**Вывод:** AI позволил мне за 2 недели сделать то, что раньше требовало команды разработчиков и нескольких месяцев.
