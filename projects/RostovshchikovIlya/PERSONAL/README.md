# PERSONAL — Мой AI-операционный контур

Я фаундер Easy Parking, не программист. AI для меня — не «ассистент для задач», а полноценный операционный контур: вместо того чтобы нанимать ресерчера, копирайтера и джуна-разработчика на старте, я закрываю эти роли скилами и плагинами в Claude Code.

Формат: `[скил] — когда триггерится — что делает — какой контекст подкладывает`

## Плагины Claude Code

Плагин = набор скилов со своими инструкциями и контекстом, устанавливается в Claude Code одной командой. У меня стоят три.

**Superpowers** — процессная дисциплина:
- `[superpowers:brainstorming]` — перед реализацией любой фичи/документа — задаёт вопросы про намерение и ограничения — не даёт лететь в код без понимания «зачем»
- `[superpowers:writing-plans]` — перед многошаговой задачей — строит план с файлами, шагами, точками проверки — защищает от хаотичной реализации
- `[superpowers:systematic-debugging]` — при любом неожиданном поведении — формулирует гипотезу → проверка → фикс — запрещает гадать
- `[superpowers:test-driven-development]` — при реализации новой логики — сначала тест, потом код — red/green/refactor

**Frontend Design** — когда строю интерфейсы:
- `[frontend-design:frontend-design]` — при любом UI — генерирует production-grade дизайн с живыми принципами, не generic AI-эстетикой — подкладывает паттерны, типографику, spacing-систему

**Feature Dev** — когда нужна фича в существующий код:
- `[feature-dev:feature-dev]` — при новой функциональности — сначала читает кодовую базу, потом предлагает архитектуру — подкладывает конвенции проекта

## Личные скилы (`.claude/commands/`)

- `[log-interview]` — после интервью с пользователем Easy Parking — превращает голосовую/текстовую расшифровку в структурированный user-interview (insight / pain / quote / segment) — подкладывает шаблон карточки инсайта и правило «один инсайт = одна карточка»
- `[weekly-review]` — каждое воскресенье — ретро недели + 3 SMART-цели на следующую — подкладывает текущие OKR и итоги прошлой недели
- `[pitch-remix]` — когда нужен питч под новый формат (холодное письмо/one-pager/акселератор) — переупаковывает общий нарратив под нужную длину и интонацию — подкладывает бренд-войс Easy Parking
- `[competitor-scan]` — при появлении нового игрока на рынке паркинга — собирает карточку конкурента (технология, ЦА, цена, слабые места) — подкладывает шаблон сравнения и наши отличия
- `[startup-answer]` — при заполнении заявок на акселераторы/гранты — пишет ответ в нужной длине и тоне — подкладывает факты о продукте, метриках, команде

## Задачи, под которые планирую собрать/найти скилы

### Meta-скилы (пригодятся в любом моём проекте)

- **Site_Monitor** — вручную обхожу сайты акселераторов и грантов в поиске открытых окон, трачу часы — хочу автоматический мониторинг списка сайтов с недельным дайджестом
- **PDF_Regulation_Extractor** — правила по телематике и парковочной политике разбросаны по PDF ЦОДД, Минтранса, ГИБДД — хочу автоматическое извлечение норм из PDF с ссылкой на источник
- **Funding_Radar** — гранты и субсидии для hardware-стартапов появляются и закрываются быстро — хочу постоянный мониторинг ФРИИ/Сколково/региональных фондов
- **Call_Summariser** — после каждого созвона с B2B-клиентом остаётся запись — хочу автоматическую выжимку «решения / обязательства / открытые вопросы / следующий шаг»
- **Competitor_Crawler** — список конкурентов в паркинге меняется, ручной мониторинг не успевает — хочу регулярный обход сайтов конкурентов с diff-отчётом
- **Market_Sizing_Assistant** — TAM/SAM/SOM считаю каждый раз заново под нового инвестора — хочу скил, который по входным гипотезам и источникам собирает расчёт

### Domain-скилы Easy Parking (поверх meta)

- **Parking_Zone_Mapper** — данные по платным зонам Москвы/СПб расползаются по сайтам — Site_Monitor + PDF_Regulation_Extractor под паркинг-данные
- **B2B_Lead_Researcher** — перед каждой встречей с таксопарком/каршерингом я вручную собираю досье компании — Claude-агент с профилем сегмента
- **Investor_Brief_Builder** — под каждого инвестора формирую свой питч-пакет — генерация деки и ответов на 20 типовых вопросов по профилю фонда
- **Customer_Story_Factory** — из логов интервью и звонков делаю истории успеха — скил для превращения сырого материала в публикуемый кейс
- **Pricing_Experimenter** — тест подписки 399₽/мес против альтернатив — скил, который моделирует unit-экономику под разные сценарии
