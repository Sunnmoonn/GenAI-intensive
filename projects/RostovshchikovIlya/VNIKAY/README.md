# VNIKAY — Учи английский, залипая в телефон

**Проект:** Вникай  
**Репозиторий:** [github.com/cuclik413/vniklay](https://github.com/cuclik413/vniklay)  
**Статус:** рабочий MVP ✅

---

## Проблема

Люди хотят учить английский, но не учат — потому что это скучно. При этом они часами смотрят короткие видео в TikTok и Reels. Существующие приложения (Duolingo, Skyeng) требуют усилия и дисциплины. Думскроллинг — не требует.

**Инсайт:** если совместить формат, которому люди уже "подсажены", с полезным контентом — барьер на вход исчезает.

---

## Решение

Вникай — веб-приложение в стиле TikTok для изучения английского. Пользователь листает короткие видео (интервью, мотивационные речи, реальные разговоры), а под каждым видео в реальном времени появляются субтитры на русском и английском — синхронно с речью.

**Никакого принуждения.** Просто листаешь — и учишься.

## Стек

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Субтитры:** Whisper (OpenAI) — расшифровка с таймингами
- **Перевод:** Claude — контекстный перевод субтитров
- **Хостинг:** GitHub

---

## Как это работает

```
Видео (MP4)
    ↓
Whisper → текст с таймингами [00:00.000 --> 00:03.500]
    ↓
Claude → перевод каждой строки на английский
    ↓
scenes.ts → массив с субтитрами в приложении
    ↓
Пользователь смотрит видео, субтитры меняются синхронно
```

**Боль:** "Я понимаю, что надо учить английский, но Duolingo надоел, а курсы дорогие и долго."
