# PROJECTCLONE — Разбор Doomersion

**Продукт:** [Doomersion](https://doomersion.com/)  
**Категория:** EdTech / Language Learning  
**Суть:** приложение для изучения иностранных языков через думскроллинг — короткие видео с субтитрами на целевом языке, подобранные под уровень пользователя.

---

## Продуктовые слои

### Слой 1 — Интерфейс (Surface)
- TikTok-подобный вертикальный плеер
- Субтитры поверх видео с побуквенной/пословной подсветкой
- Кнопки: лайк, "слишком сложно / слишком просто", словарь
- Мобильное приложение (iOS / Android)

### Слой 2 — Контентный слой
- Короткие видео (реальный контент носителей языка: блогеры, новости, подкасты)
- Субтитры с таймингами на целевом языке + перевод
- Разбивка слов на морфемы, транскрипция, примеры употребления

### Слой 3 — Персонализация и адаптация
- Определение уровня пользователя (A1–C2)
- Рекомендательная система: подбирает видео по уровню и интересам
- Адаптивная сложность: если пользователь говорит "слишком сложно" — контент упрощается

### Слой 4 — Монетизация
- Freemium: базовый доступ бесплатно, расширенный словарь и офлайн — платно
- Подписка (~$10/мес)

### Слой 5 — Данные
- Поведенческие данные: что пользователь смотрит, что пропускает, где ставит на паузу
- Словарный прогресс: какие слова встречались, сколько раз, в каком контексте

---

## Где живёт moat (защитный ров)

| Слой | Сложность копирования | Почему |
|------|-----------------------|--------|
| UI/UX плеера | 🟡 Средняя | Интерфейс воспроизводим, но нужна полировка |
| Контент-библиотека | 🔴 Высокая | Тысячи размеченных видео с субтитрами = месяцы работы |
| Рекомендательная система | 🔴 Высокая | Обучается на данных пользователей — чем больше юзеров, тем лучше |
| Словарная база | 🟡 Средняя | Можно собрать из открытых источников, но контекстная привязка — сложно |
| Пользовательские данные | 🔴 Высокая | Накопленный прогресс невозможно перенести — lock-in |

**Главный moat** — рекомендательная система + накопленный словарный прогресс пользователя. Чем дольше человек использует приложение, тем точнее подбор и тем сложнее уйти (потерять прогресс).

## Что можно улучшить / где слабые места

1. **Контент на русском языке** — Doomersion ориентирован на английский рынок, русскоязычная аудитория недообслужена
2. **Нет веб-версии** — только мобильное приложение
3. **Субтитры только на целевом языке** — нет режима "перевод рядом" для начинающих
4. **Онбординг слабый** — уровень определяется приблизительно

## Вывод

Doomersion — умный продукт с правильной гипотезой: люди уже залипают в короткие видео, нужно просто направить это в обучение. Главная ставка — на рекомендации и данные. MVP воспроизводим за недели, но настоящий продукт строится месяцами через накопление контентной базы и поведенческих данных.
