# КИН / kin-watch

Сервис информирования близких о состоянии пожилого человека на основе данных смартфона и носимых устройств. Система отслеживает активность, сон и повседневные паттерны, замечает тревожные отклонения и заблаговременно уведомляет семью.

## Суть продукта

Когда пожилой родственник живёт отдельно, семья находится в постоянном фоновом беспокойстве. Ежедневные звонки — ненадёжный способ контроля, камеры — инвазивны, кнопки тревоги — реактивны.

КИН собирает данные с привычных устройств пожилого человека, строит персональный поведенческий базовый профиль и замечает отклонения раньше, чем ситуация становится критической. Семья получает спокойствие без постоянного ручного мониторинга.

**Ключевые роли:**
- **Плательщик / инициатор** — дочь или сын, 30–55 лет
- **Пользователь** — пожилой родственник (пассивный, минимальные усилия)
