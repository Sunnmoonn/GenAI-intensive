# CODEBASE

Описание кодовой базы проекта РЯДОМ.

**Основной репозиторий:** [yarochkinama/kin-watch](https://github.com/yarochkinama/kin-watch)

---

## Что находится в репозитории

### Лендинг

- Собран на **Replit**
- Текущая версия включает: описание продукта, форму сбора контактов / предзаписи, блок с ценностным предложением
- На текущем этапе лендинг выполняет роль инструмента валидации спроса, а не production frontend

### ML-прототип

- Базовый прототип детекции изменений активности
- Разработан на **открытых данных** (HAR dataset — Human Activity Recognition)
- Подтверждает гипотезу: паттерны снижения активности можно выявить раньше, чем произойдёт явное ухудшение
- Статус: proof-of-concept, не production-ready
- Следующий шаг: проверка подхода на реальных обезличенных данных

## Примечание

На текущей стадии проект находится в фазе product discovery и валидации гипотез. Кодовая база отражает это: приоритет — скорость проверки, а не масштабируемость. Архитектурные решения для production-версии будут приняты после подтверждения подхода на реальных данных.
