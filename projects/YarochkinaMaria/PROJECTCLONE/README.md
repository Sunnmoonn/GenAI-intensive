> Домашнее задание по курсу GenAI Intensive — клонирование реального продукта с помощью AI-инструментов.

Для клонирования был выбран **[Settld](https://settld.care)** — британский сервис уведомления организаций о смерти. Реализована ключевая часть продукта: создание кейса, выбор организаций, отслеживание статусов запросов и портал для организаций.

Цель задания — пройти полный цикл от анализа сервиса до работающего MVP с помощью AI-skills: выбор продукта, JTBD-анализ, декомпозиция на модули, разработка фронтенда, публикация.

## 🚀 MVP-клон: Settld

Settld решает одну болезненную проблему: когда умирает близкий, семье нужно уведомить десятки организаций — банки, коммунальные службы, телеком, страховые. Вручную это занимает месяцы.

**Реализованный сценарий:** семья создаёт кейс → выбирает организации → получает сводный дашборд с прогрессом по каждому запросу. Организации видят входящие уведомления и меняют статусы через свой портал.

**Что работает в демо:**
- Мастер создания кейса (3 шага)
- Выбор организаций из реестра с поиском и категориями
- Дашборд с круговым прогрессом и статистикой
- Детальный просмотр каждого запроса с историей статусов
- Портал организации — смена статусов по state machine

**Demo:** [settld-app.vercel.app](https://settld-app.vercel.app)

## 🛠 Что реализовано

- **Экран выбора роли** — инициатор (семья) или организация, кнопка демо-режима
- **Мастер создания кейса** — 3 шага: данные умершего, данные заявителя, подтверждение
- **Выбор организаций** — реестр из 19 организаций, поиск, фильтр по 5 категориям
- **Дашборд кейса** — прогресс-кольцо, статистика по статусам, список запросов с цветовой индикацией
- **Детали запроса** — обратный timeline статусов с временными метками и заметками
- **Портал организации** — входящие запросы, фильтр по org и статусу, смена статусов
- **State machine** — 6 статусов (PENDING → SENT → ACKNOWLEDGED → PROCESSING → COMPLETED / REJECTED)
- **Демо-данные** — кейс David Harrison, 8 запросов в разных статусах для наглядности
- **Mobile-first UI** — max-width 430px, Inter, тёмно-синяя палитра (#1C3D5E)
