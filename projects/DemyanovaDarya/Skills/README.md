# Skills

Набор инструментов, которые используются в повседневной работе над проектами.

---

## Скиллы общего назначения

### `presentation` — создание презентаций

Подготовка структурированных презентаций: питчи для инвесторов, учебные материалы, отчёты. Помогает выстроить логику повествования и адаптировать подачу под аудиторию. Поддерживает несколько визуальных стилей. На выходе — готовый HTML-файл, открывается в браузере без установки программ.

### `pdf-to-md` — конвертация документов в Markdown

Преобразует PDF, PPTX, DOCX, XLSX и изображения в чистый Markdown через MarkItDown (Microsoft). Сохраняет структуру заголовков, таблицы, списки. Устанавливается автоматически при первом запуске.

### `img` — генерация изображений

Создание изображений по текстовому описанию через kie.ai и аналоги. Используется для визуалов к презентациям и иллюстраций. Поддерживает несколько моделей генерации.

### `caveman` — режим лаконичности

Переключает стиль ответов в сжатый формат: без вводных фраз, без воды, только суть. Несколько уровней интенсивности. Отключается командой `stop caveman`.

### `deploy` — публикация на GitHub и Vercel ⭐ доп. балл

Полный цикл от локального проекта до живого сайта с автодеплоем. Покрывает git, GitHub, Vercel, переменные окружения, работу с несколькими аккаунтами и токенами. Включает решения типичных ошибок. Разработан для получения дополнительного балла (оценка 5).

### `planner` — расписание из Google Таблицы в Apple Notes

Читает Google Таблицу с расписанием, извлекает ключевые события и автоматически создаёт структурированную заметку в приложении Заметки на Mac — по дням и часам.

## Скиллы проекта Объяснятор

Следующие скиллы описывают ключевые операции внутри проекта и могут использоваться самостоятельно вне веб-интерфейса.

### `chunker` — разбивка текста на фрагменты

Разбивает длинный документ на смысловые чанки по заголовкам и абзацам с fallback на фрагменты по 800 символов с перекрытием. Включает keyword scoring для поиска релевантных фрагментов по вопросу пользователя. Используется перед передачей любого большого текста в языковую модель.

### `quiz` — генерация квиза

Создаёт 5 вопросов с вариантами ответов A/B/C/D и объяснением правильного ответа строго по переданному тексту. Возвращает структурированный JSON. Используется в Объяснятор для проверки знаний по загруженному документу.

### `flashcards` — генерация флеш-карточек

Создаёт 8–10 пар «термин — определение» из любого текста. Удобно для запоминания понятий из лекций и учебников. В веб-версии карточки отображаются с анимацией переворота.

### `practice` — генерация практических задач

Создаёт 3 задачи на применение знаний из текста с двумя прогрессивными подсказками и полным решением. Пользователь сначала пробует решить сам, затем запрашивает подсказки по одной.
