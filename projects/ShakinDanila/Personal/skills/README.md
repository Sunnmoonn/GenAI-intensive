# Скиллы Данилы Шакина для Claude Code

Коллекция персональных скиллов для Claude Code. Установка: скопировать папку скилла в `~/.claude/skills/` — скилл подхватывается автоматически без перезапуска.

---

## 📋 Список скиллов

### `/тз` — Техническое Задание
**Что делает:** Составляет чёткое и практичное ТЗ для технических проектов. Не ГОСТ-бюрократия, а живой документ для команды.

**Поддерживает:** аппаратные стенды, веб-приложения, мобильные приложения, AI/ML-системы, автоматизацию, интеграции.

**Как использовать:**
```
/тз
составь ТЗ на проект
напиши техническое задание для [описание]
```

**Что получишь:** структурированный markdown-файл с разделами: обзор, цели и метрики, пользователи и сценарии, функциональные требования, архитектура, стек, интеграции, нефункциональные требования, этапы.

---

### `/план` — Календарный план с часами и бюджетом
**Что делает:** Составляет календарный план реализации с распределением часов по ролям, бюджетом по этапам и помесячной разбивкой.

**Как использовать:**
```
/план
составь план реализации
календарный план с бюджетом для [описание проекта]
```

**Что получишь:** таблица этапов (длительность, часы по ролям PM/Dev/QA, бюджет) + описание каждого этапа с Deliverable + KPI и критерии приёмки + реестр рисков + контрольные точки + помесячный бюджет.

**Пример:** для проекта 2 человека × 40 ч/нед × 10 месяцев выдаст 10 этапов с распределением часов и бюджетом 1 000 000 руб.

---

### `/оценка` — Оценка сложности проекта
**Что делает:** Оценивает сложность проекта по 5 осям и выдаёт уровень, трудозатраты, риски и рекомендации.

**Как использовать:**
```
/оценка
оцени проект [описание]
насколько сложно сделать [описание]
```

**5 осей оценки:** техническая сложность / объём работ / неопределённость / интеграции / команда

**Уровни:** 🟢 Просто → 🔵 Стандарт → 🟡 Серьёзно → 🟠 Сложно → 🔴 Монстр

**Что получишь:** балл по каждой оси → диапазон трудозатрат с буфером → топ-3 риска → рекомендации → стоп-сигналы.

---

### `/заказчику` — Сообщение заказчику
**Что делает:** Переводит неформальный текст в профессиональное деловое сообщение. Пишешь как зумер — получаешь как директор.

**Как использовать:**
```
/заказчику
напиши заказчику: [мысль в свободной форме]
переведи заказчику: чутка задержимся, нашли баг
```

**Форматы:** чат (Telegram/Slack), email (с темой и подписью), комментарий в трекере.

**Пример:**
- Вход: «скажи им что мы чутка задержимся, там баг нашли короч, к пятнице точно сделаем»
- Выход: «В ходе финального тестирования был выявлен технический дефект, требующий дополнительной проработки. Срок сдачи незначительно смещается — готовность подтверждаем к пятнице включительно.»

---

### `/презентация` — HTML-презентация на Reveal.js
**Что делает:** Генерирует и редактирует красивые HTML-презентации из текста, заметок или темы. Файл открывается в браузере без установки.

**Как использовать:**
```
/презентация
сделай презентацию на тему [тема]
упакуй это в слайды: [текст/ТЗ/план]
поменяй стиль на Dark Mode
```

**Стили:** Minimalism (питч-дек) / Dark Mode (техно) / Neobrutalism (конференция) / Avant (креатив)

**Форматы:** 12 типов слайдов, Mermaid.js диаграммы, Chart.js графики, speaker notes, PDF-экспорт.

---

### `pdf-to-md` — Конвертация документов в Markdown
**Что делает:** Конвертирует PDF, DOCX, PPTX, XLSX, HTML, изображения в чистый Markdown через MarkItDown (Microsoft).

**Как использовать:**
```
конвертируй файл в markdown: [путь к файлу]
переведи в md: C:/path/to/document.pdf
```

**Что получишь:** `.md` файл рядом с исходником или в указанной папке.

---

### `изобразика` — Генерация изображений
**Что делает:** Генерирует изображения через kie.ai API. Все модели GEN Studio.

**Как использовать:**
```
нарисуй [описание]
сгенерируй изображение: [промпт]
создай иллюстрацию для слайда
```

**Модели:** Nano Banana, Flux, Seedream, Ideogram, GPT-4o Image, Z-Image, Grok.

---

### `caveman` — Режим экономии токенов
**Что делает:** Сжимает ответы Claude на ~75% токенов, говорит как пещерный человек, но технически точно.

**Как использовать:**
```
caveman mode
/caveman
/caveman ultra
stop caveman
```

**Уровни:** lite (без воды, полные предложения) / full (классический кавемен) / ultra (максимальное сжатие) / wenyan-* (классический китайский)

**Пример:** «New object ref each render. Inline object prop = new ref = re-render. Wrap in `useMemo`.»

---

### `полный-доступ` — Автономный режим
**Что делает:** Активирует режим полного доступа — Claude действует без запроса подтверждений на любые действия на компьютере.

**Исключение:** Telegram (любые действия запрещены).

**Как использовать:**
```
полный доступ
/полный-доступ
действуй свободно
```

---

## 🚀 Установка

```bash
# Скопировать нужный скилл
cp -r skills/тз ~/.claude/skills/тз

# Или все сразу
cp -r skills/* ~/.claude/skills/
```

После копирования скилл подхватывается автоматически (перезапуск не нужен).
