# Скилл: Firecrawl Market Research

## Триггер

Когда нужно собрать данные о рынке, конкурентах или аудитории из открытых источников — без ручного серфинга.

## Что делает

Использует Firecrawl для:
- Скрейпинга сайтов конкурентов (pricing, features, positioning)
- Поиска статей и отчётов по рынку (TAM/SAM/SOM источники)
- Извлечения структурированных данных (таблицы, списки) из сложных страниц

## Применение

1. **Анализ конкурента по URL** → скрейп главной + pricing + docs → конкурентная матрица
2. **Поиск рыночных данных** → `firecrawl search "ELN LIMS market size 2025"` → агрегация источников для TAM
3. **Мониторинг** → периодический скрейп страниц конкурентов на изменения

## Контекст, который нужно подложить

- URL или поисковый запрос
- Что именно нужно извлечь (pricing / features / market data)
- Формат вывода (таблица / markdown / JSON)

## Пример применения

Сбор данных о конкурентах Квантары (Benchling, LabWare, 1С:LIMS, myLab) для конкурентной матрицы в PRODUCT.md.
