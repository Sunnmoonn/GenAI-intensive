# Скилл: Clone Step 1 — Поиск сервиса под клонирование

## Триггер

Когда нужно найти зарубежный сервис для клонирования под российский рынок из длинного списка кандидатов.

## Что делает

Систематически сужает пространство поиска от сотен стартапов до одного финального кандидата через количественный скоринг и ручной фильтр.

## Шаги

1. Определить нишу поиска: вертикаль, тип клиента (B2B/B2C), наличие санкционных барьеров для западных игроков
2. Собрать кандидатов: YC batch за последний год, Product Hunt топ по нише, Crunchbase поиск по тегам
3. Применить VENTURE-SCAN RF 2.0 скоринг: `Score = (M² × T × A × B × P)^(1/6) × 10` — **M** (ёмкость рынка РФ), **T** (технологичность), **A** (применимость), **B** (барьеры для западных), **P** (потенциал партнёрств)
4. Отфильтровать топ-10 по скору, проверить вручную: есть ли боль у российских пользователей, есть ли аналог с полным функционалом
5. Зафиксировать финальный выбор + критерии отсева остальных (таблица)

## Критерии отсева кандидата

- Доступен в РФ и активно продаётся → не нужно клонировать
- Сильный отечественный аналог с тем же функционалом уже существует
- Рынок РФ < ₽300 млн ARR → не стоит усилий
- UI/UX принципиально не воспроизводим без патентованных технологий

## Контекст для подкладки

- Ниша / вертикаль (например: ELN+LIMS, HR-tech, LegalTech)
- Ограничения стека (если есть)
- Горизонт: MVP за 2 недели или полноценный продукт

## Пример

Ниша: ELN+LIMS → 22 оценённых сервиса → итог: Benchling (единственный с полным стеком ELN+LIMS+AI, заблокирован санкциями, воспроизводимый React-интерфейс).
